# Glossary

## Phronesis
Greek concept of practical wisdom. Project name.

## Pipecat
Open-source Python framework for building voice agents. Handles orchestration of STT, LLM, TTS, transport, and pipeline logic.

## Modal
Serverless AI/GPU infrastructure platform. Provides cloud GPUs and Python-first deployment.

## MisoTTS
Open-weight text-to-speech model from Miso Labs. Used for natural voice generation.

## Deepgram Nova-3
Speech-to-text service selected for high-quality real-time transcription.

## MiniMax M3
Open-weight, long-context, agentic LLM served via Ollama Cloud. Selected as primary reasoning model.

## Pinecone
Managed vector database used for document retrieval and long-term memory storage.

## LangSmith
Observability and tracing platform for LLM and agentic workflows.

## RAG
Retrieval-Augmented Generation. The system retrieves relevant documents and injects them into the prompt.

## Chunk
A stored retrieval unit. For this project, chunks are full pages.

## Math Object
Structured object containing both spoken and display versions of a mathematical expression.

## Tutor Mode
Initial operating mode focused on exam preparation, pain points, and rigorous intuition.

## Vibe Engineering
Iterative development workflow using coding agents, traces, small scopes, and taste-driven evaluation.
