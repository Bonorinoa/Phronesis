# Phronesis

**Phronesis** is a phone-first voice study companion for rigorous conversations about economic theory and recursive macroeconomic modeling.

It began as a Brain Fart on 2026-06-14 and is now graduated into an active Tool Project.

## Core Idea

Build a tasteful, highly capable conversational tutor: attentive, curious, rigorous, witty, challenging, and knowledgeable. It should feel like the kind of intellectual friend or mentor one would want to walk with, argue with, and learn from.

The initial constraint is macroeconomics qualifier preparation, but the design should generalize to advanced technical conversation in economics and adjacent fields.

## Project Priorities

1. Natural, non-robotic voice generation
2. Technical accuracy on economic theory and rigorous intuition on recursive macroeconomic modeling
3. Low friction: works from phone, no laptop dependency
4. Beautiful, thoughtful, tasteful interface
5. Useful memory and context management
6. Export conversations into learning artifacts
